About the Project

Test Automation Report - Summary of the project

Clinical Trial Knowledge Management System (CTKMS) or Knowledge Management System (KMS) is a single, central maintained repository with defined data owners that holds key country clinical trial information.

The system ensures data remains up-to-date and compliant with country-specific legislation

The project features two distinct portals:

Admin Portal: Used for managing configurations and overseeing data control.

End-User Portal: Designed for easy access to reference clinical trial information.

KMS is built on a serverless microservices architecture and is fully deployed on AWS, ensuring scalability, cost-efficiency, and ease of maintenance.


KMS can host multiple workspaces in it. Repository of Clinical trials Knowledge (ROCK) is one such workspace.
